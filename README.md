Jag har förbättrat sidan så att den blir mer tillgänglig och lättare att använda. I HTML fixade jag strukturen genom att använda rätt element, ta bort dubbla saker och lägga till labels på formulär samt alt-text på bilder. Jag gjorde också länkar och knappar tydligare.

I CSS gjorde jag designen enklare och mer läsbar. Jag tog bort konstiga effekter, minskade textstorleken, fixade färger och kontrast och gjorde layouten mer stabil. Jag tog även bort störande saker som animationer och den fasta annonsen.

I JavaScript tog jag bort kod som gjorde sidan långsam och jobbig, till exempel loopar, document.write() och kod som ändrade bakgrunden hela tiden. Jag gjorde scriptet enklare och mer relevant.

Reflektion

Det som var svårt var att många problem inte syns direkt utan ligger i koden. Jag lärde mig att tillgänglighet handlar mycket om struktur, tydliga texter och att undvika onödiga effekter. Jag förstod också att enkel design ofta är bättre och att både HTML, CSS och JavaScript påverkar hur lätt sidan är att använda.
